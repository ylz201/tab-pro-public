# Tab Pro

Tab Pro is a local-first Chrome extension for fast tab switching, duplicate tab cleanup, stale tab cleanup, and local session snapshots.

It is built for browser-heavy workflows where users need to quickly find, switch, group, and clean up open tabs without sending browsing data to a cloud service.

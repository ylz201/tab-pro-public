# Tab Pro

Tab Pro is a local-first Chrome extension for fast tab switching, duplicate tab cleanup, stale tab cleanup, and local session snapshots.

It is designed for people who keep many browser tabs open during research, operations, development, writing, shopping, or daily knowledge work.

## What Tab Pro Does

- Open a keyboard-first popup for fast tab search and switching.
- Jump to ranked tabs with number keys.
- Search tabs by title, URL, domain, pinned state, audible state, duplicate state, and stale state.
- Review tabs in timeline and grouped views.
- Clean up duplicate tabs while keeping the most useful copy.
- Find and close stale tabs with an undo path.
- Save local session snapshots for later restore.
- Import and export local session data.
- Configure grouping rules for personal workflows.

## Privacy Model

Tab Pro is built as a local-first browser tool.

- No account is required for core tab management.
- No telemetry is built into the product website.
- Tab metadata and settings are stored locally in the user's Chrome profile.
- Payment processing, when used, is handled by third-party payment processors such as PayPal.

## Frequently Asked Questions

### Is this the source code repository?

No. This repository is for public product information and support only. It does not contain Tab Pro source code, extension packages, license logic, payment implementation, or commercial implementation details.

### Why is the source code not in this repository?

Tab Pro is distributed as a browser extension product. Public support, product information, privacy policy, refund policy, and checkout information are kept here so users and payment reviewers can verify the product without exposing private implementation details.

### Does Tab Pro replace the new tab page?

No. Tab Pro works as a Chrome extension popup and settings page. It does not need to replace the browser's new tab page.

### Does Tab Pro send my tab data to a server?

No cloud workspace or telemetry service is required for core tab management. Tab Pro uses Chrome extension APIs and local Chrome storage for tab workflows.
